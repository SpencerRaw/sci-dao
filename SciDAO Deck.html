SciDAO Architecture

Layer 1

🧬 Curiosity Engine

AI explores domains, generates hypotheses, designs experiments. 3-strategy generation: literature-grounded, scientific debate, iterative assumptions.

→
Layer 2

🔧 Task Marketplace

Hypotheses atomized into executable units. Each task: materials, steps, time estimate, difficulty. Anyone can claim and execute.

→
Layer 3

Immutable Contribution Graph

// Every contribution is permanently recorded ledger.record( contributor: "Alice Chen", task: "TASK-HYP-001-03", hypothesis: "HYP-001", content: "MTT assay: HeLa + 50μg/ml NIR-CD", weight: 0.15 )
// Verifiable integrity ledger.verify("ENT-a3f2b9c1") // → true: SHA-256 matches stored hash
// Granular attribution leaderboard: 1. Alice Chen 0.450 (3 tasks) 2. Bob Wang 0.300 (2 tasks) 3. Carol Li 0.150 (1 task)

🔐 Content Hashing

SHA-256 hash of every submission. Tamper-evident. No blockchain needed — pure cryptographic integrity.

⚖️ Granular Weight

Contributions weighted 0.0–1.0. A single data point = 0.15. A breakthrough insight = 0.50. PI doesn't monopolize credit.

🔗 Full Lineage

Every entry links to its parent. Trace any result back through the entire chain of contributions.
